Part 1: Extracting Human rRNA Sequences from SILVA (18S/28S)
SILVA provides comprehensive rRNA databases for many species. To focus on human sequences:

Download full eukaryotic 18S and 28S fasta files from SILVA:

bash
wget https://www.arb-silva.de/fileadmin/silva_databases/release_138.2/Exports/SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta.gz

wget https://www.arb-silva.de/fileadmin/silva_databases/release_138.2/Exports/SILVA_138.2_LSURef_NR99_tax_silva_trunc.fasta.gz


gunzip SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta.gz
gunzip SILVA_138.2_LSURef_NR99_tax_silva_trunc.fasta.gz

confirm the presence
grep -i "Homo sapiens" -A 1 SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta | head -20
grep -i "Homo sapiens" -A 1 SILVA_138.2_LSURef_NR99_tax_silva_trunc.fasta | head -20


and extract human sequences by filtering fasta headers containing "Homo sapiens" or "human":
conda install -c bioconda seqkit
grep -i "Homo sapiens (human)" SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta | cut -d ' ' -f1 > headers.txt
grep -i "^>" SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta | grep -i "Homo sapiens" | sed 's/>//' | cut -d ' ' -f1 > headers_18s.txt

grep -i "^>" SILVA_138.2_LSURef_NR99_tax_silva_trunc.fasta | grep -i "Homo sapiens" | sed 's/>//' | cut -d ' ' -f1 > headers_28s.txt

extract using seqtk
conda install -c bioconda seqtk
seqtk subseq SILVA_138.2_SSURef_NR99_tax_silva_trunc.fasta headers_18s.txt > silva-euk-18s-human.fasta

seqtk subseq SILVA_138.2_LSURef_NR99_tax_silva_trunc.fasta headers_28s.txt > silva-euk-28s-human.fasta


check stats
seqkit stats silva-euk-18s-human.fasta silva-euk-28s-human.fasta


Part 2: Getting 5S and 5.8S rRNA from Rfam
Rfam holds RNA families including 5S and 5.8S rRNA sequences.

Download Rfam full fasta and extract:

https://rfam.xfam.org/family/RF00001/alignment
https://rfam.xfam.org/family/RF00002/alignment
save these locally as text file(save as .txt)

to convert .txt to .fasta(use this py script)
import sys

input_file = sys.argv[1]
output_file = sys.argv[2]

with open(input_file) as f, open(output_file, 'w') as out:
    seqs = {}
    for line in f:
        line = line.strip()
        if not line or line.startswith("#") or line.startswith("//"):
            continue
        if line.startswith(" "):  # skip consensus lines
            continue
        parts = line.split()
        if len(parts) == 2:
            seq_id, seq = parts
            seq = seq.replace("-", "")  # remove gaps
            if seq_id in seqs:
                seqs[seq_id] += seq
            else:
                seqs[seq_id] = seq
    for seq_id, seq in seqs.items():
        out.write(f">{seq_id}\n{seq}\n")
        
sto2fasta.py file 

python sto2fasta.py Rf00001.txt rfam-5s.fasta
python sto2fasta.py Rf00002.txt rfam-5.8s.fasta
